If you want to create a CHECK constraint to ensure that the value in the Department column is always 'IT', you can do this at either the column level or the table level.

Column-Level CHECK Constraint
Here’s how to create a table with a column-level CHECK constraint that enforces that the Department column must always be 'IT':

sql
Copy code
CREATE TABLE Employees (
    EmployeeID INT PRIMARY KEY,
    FirstName NVARCHAR(50),
    LastName NVARCHAR(50),
    Department NVARCHAR(50) CHECK (Department = 'IT')
);
Table-Level CHECK Constraint
Here’s how to create a table with a table-level CHECK constraint that enforces that the Department column must always be 'IT':

sql
Copy code
CREATE TABLE Employees (
    EmployeeID INT PRIMARY KEY,
    FirstName NVARCHAR(50),
    LastName NVARCHAR(50),
    Department NVARCHAR(50),
    CONSTRAINT CHK_Department CHECK (Department = 'IT')
);
